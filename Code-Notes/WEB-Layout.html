    # CSS flex布局

    # 优点：

    • 在元素外部处理空间分布
    • 让元素自己处理空间分布

    # 适用场景：

    • 导航
    • 拆分导航
    • 元素居中
    • 绝对底部

    # Flex容器属性

    • flex-direction（排列方向）

      沿水平方向从左向右、从右向左

      沿垂直方向的正向、逆向

    • flex-wrap（换行）

      元素超出空间，挤压在主轴上的大小

    • flex-flow（前两者一并）

    # Flex元素属性

    • flex-basis

      控制flex元素在主轴方向上所占的大小，没指定时默认为width的大小为最小大小

    • flex-grow

      对容器剩余空间的处理
