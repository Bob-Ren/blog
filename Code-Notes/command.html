    # you-get安装与使用

    # 安装命令

    brew install you-get
    1

    # 下载视频方式

    首先输入如下命令，获取不同格式、清晰度的视频列表。列表中参数itag为视频标记序号，下面的下载命令通过itag这个参数来告诉工具你要下载的清晰度。

    # 命令
you-get -i 'url'
# 示例
you-get -i 'https://www.youtube.com/watch?v=4nv0QtHGWXw&list=PLRMOX8QaZK8xWg64nF5JgmD3T1UccCRfG&index=1&t=76s'
    1
    2
    3
    4

    比如我要下载列表中第一个1280 x 720p的视频，可以看到对应视频的itag为136，输入如下指令后即可下载：

    you-get --itag=136 'https://www.youtube.com/watch?v=4nv0QtHGWXw&list=PLRMOX8QaZK8xWg64nF5JgmD3T1UccCRfG&index=1&t=76s'
    1

    使用"--playlist"语句下载整个YouTube视频列表

    you-get --itag=136 --playlist 'https://www.youtube.com/watch?v=4nv0QtHGWXw&list=PLRMOX8QaZK8xWg64nF5JgmD3T1UccCRfG&index=1&t=76s'
    1

    注：Mac的终端使用的是zsh，需要想上方例子一样在链接参数两侧加上单引号。其他终端如Windows不需要给链接加单引号。

    # 下载路径

    可以在终端中使用cd命令进入你想要下载到的文件夹，在该目录下使用you-get命令进行下载默认就会储存到当前文件夹中。

    # V2RayX代理系统终端（Mac）

    编辑".zshrc"文件，添加两行代码修改指令别名：

    alias proxy='export all_proxy=socks5://127.0.0.1:1080'
alias unproxy='unset all_proxy'
    1
    2

    终端执行

    source ~/.zshrc
    1

    之后就可以使用命令proxy、unproxy开启和关闭代理

    另外可以添加如下行继承".bash_profile"中的配置

    source ~/.bash_profile
    1
