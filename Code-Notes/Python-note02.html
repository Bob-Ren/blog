    # 字符串内建函数

    序号 方法及描述
    1 capitalize() (opens new window) 将字符串的第一个字符转换为大写
    2 center(width, fillchar) (opens new window)返回一个指定的宽度 width 居中的字符串，fillchar 为填充的字符，默认为空格。
    3 count(str, beg= 0,end=len(string)) (opens new window) 返回 str 在 string 里面出现的次数，如果 beg 或者 end 指定则返回指定范围内 str 出现的次数
    4 bytes.decode(encoding="utf-8", errors="strict") (opens new window) Python3 中没有 decode 方法，但我们可以使用 bytes 对象的 decode() 方法来解码给定的 bytes 对象，这个 bytes 对象可以由 str.encode() 来编码返回。
    5 encode(encoding='UTF-8',errors='strict') (opens new window) 以 encoding 指定的编码格式编码字符串，如果出错默认报一个ValueError 的异常，除非 errors 指定的是'ignore'或者'replace'
    6 endswith(suffix, beg=0, end=len(string)) (opens new window) 检查字符串是否以 obj 结束，如果beg 或者 end 指定则检查指定的范围内是否以 obj 结束，如果是，返回 True,否则返回 False.
    7 expandtabs(tabsize=8) (opens new window) 把字符串 string 中的 tab 符号转为空格，tab 符号默认的空格数是 8 。
    8 find(str, beg=0, end=len(string)) (opens new window) 检测 str 是否包含在字符串中，如果指定范围 beg 和 end ，则检查是否包含在指定范围内，如果包含返回开始的索引值，否则返回-1
    9 index(str, beg=0, end=len(string)) (opens new window) 跟find()方法一样，只不过如果str不在字符串中会报一个异常。
    10 isalnum() (opens new window) 如果字符串至少有一个字符并且所有字符都是字母或数字则返 回 True，否则返回 False
    11 isalpha() (opens new window) 如果字符串至少有一个字符并且所有字符都是字母或中文字则返回 True, 否则返回 False
    12 isdigit() (opens new window) 如果字符串只包含数字则返回 True 否则返回 False..
    13 islower() (opens new window) 如果字符串中包含至少一个区分大小写的字符，并且所有这些(区分大小写的)字符都是小写，则返回 True，否则返回 False
    14 isnumeric() (opens new window) 如果字符串中只包含数字字符，则返回 True，否则返回 False
    15 isspace() (opens new window) 如果字符串中只包含空白，则返回 True，否则返回 False.
    16 istitle() (opens new window) 如果字符串是标题化的(见 title())则返回 True，否则返回 False
    17 isupper() (opens new window) 如果字符串中包含至少一个区分大小写的字符，并且所有这些(区分大小写的)字符都是大写，则返回 True，否则返回 False
    18 join(seq) (opens new window) 以指定字符串作为分隔符，将 seq 中所有的元素(的字符串表示)合并为一个新的字符串
    19 len(string) (opens new window) 返回字符串长度
    20 [ljust(width, fillchar]) (opens new window) 返回一个原字符串左对齐,并使用 fillchar 填充至长度 width 的新字符串，fillchar 默认为空格。
    21 lower() (opens new window) 转换字符串中所有大写字符为小写.
    22 lstrip() (opens new window) 截掉字符串左边的空格或指定字符。
    23 maketrans() (opens new window) 创建字符映射的转换表，对于接受两个参数的最简单的调用方式，第一个参数是字符串，表示需要转换的字符，第二个参数也是字符串表示转换的目标。
    24 max(str) (opens new window) 返回字符串 str 中最大的字母。
    25 min(str) (opens new window) 返回字符串 str 中最小的字母。
    26 [replace(old, new , max]) (opens new window) 把 将字符串中的 old 替换成 new,如果 max 指定，则替换不超过 max 次。
    27 rfind(str, beg=0,end=len(string)) (opens new window) 类似于 find()函数，不过是从右边开始查找.
    28 rindex( str, beg=0, end=len(string)) (opens new window) 类似于 index()，不过是从右边开始.
    29 [rjust(width,, fillchar]) (opens new window) 返回一个原字符串右对齐,并使用fillchar(默认空格）填充至长度 width 的新字符串
    30 rstrip() (opens new window) 删除字符串末尾的空格或指定字符。
    31 split(str="", num=string.count(str)) (opens new window) 以 str 为分隔符截取字符串，如果 num 有指定值，则仅截取 num+1 个子字符串
    32 [splitlines(keepends]) (opens new window) 按照行('\r', '\r\n', \n')分隔，返回一个包含各行作为元素的列表，如果参数 keepends 为 False，不包含换行符，如果为 True，则保留换行符。
    33 startswith(substr, beg=0,end=len(string)) (opens new window) 检查字符串是否是以指定子字符串 substr 开头，是则返回 True，否则返回 False。如果beg 和 end 指定值，则在指定范围内检查。
    34 [strip(chars]) (opens new window) 在字符串上执行 lstrip()和 rstrip()
    35 swapcase() (opens new window) 将字符串中大写转换为小写，小写转换为大写
    36 title() (opens new window) 返回"标题化"的字符串,就是说所有单词都是以大写开始，其余字母均为小写(见 istitle())
    37 translate(table, deletechars="") (opens new window) 根据 str 给出的表(包含 256 个字符)转换 string 的字符, 要过滤掉的字符放到 deletechars 参数中
    38 upper() (opens new window) 转换字符串中的小写字母为大写
    39 zfill (width) (opens new window) 返回长度为 width 的字符串，原字符串右对齐，前面填充0
    40 isdecimal() (opens new window) 检查字符串是否只包含十进制字符，如果是返回 true，否则返回 false。

    # 列表推导式

    两种推导式格式
    [表达式 for 变量 in 列表]
    [out_exp_res for out_exp in input_list]
    [表达式 for 变量 in 列表 if 条件]
    [out_exp_res for out_exp in input_list if condition]
    • out_exp_res: 列表生成元素表达式，可以有返回值
    • for_out_exp in input_list: 迭代input_list将out_exp传入out_exp_res表达式中
    • if condition: 条件语句，可过滤列表中不合适的值
    # 过滤长度小于等于3的字符串，并将剩下的转换为大写字母
names = ['bob', 'tom', 'alice', 'peter', 'smith']
new_name = [name.upper() for name in names if len(name)>3]
print(new_name)     # 输出['ALICE', 'PETER', 'SMITH']

# 计算30以内可以被3整除的数
multiples = [i for i in range(30) if i % 3 == 0]
print(multiples)             # 输出[0, 3, 6, 9, 12, 15, 18, 21, 24, 27]
