    轻松使用VuePress框架搭建个人博客

    Bob.Ren 2/9/2022 VuePress

    # 环境准备

    • 安装node.js 、npm

    我使用的node.js版本是16.14，17在搭建过程中遇到了一些问题没有解决，换成16就完全正常了，所以建议大家目前还是安装版本16。另外node.js最近的几个版本中都内置了npm包管理器，所以就不用另外安装了。

    下载地址：Download | Node.js (nodejs.org) (opens new window)

    下载对应你电脑系统的安装程序，安装后在终端执行这两条命令查询node.js和npm版本号，正确显示即安装成功

    # node.js版本号查看
node -v

# npm版本号查看
npm -v

    # 本地搭建

    # 相关文件配置

    1. 选择一个目录存放项目的本地文件，比如我选择在桌面上新建一个名为“blogdemo”的文件夹作为项目根目录。使用cd命令进入文件
    cd desktop/blogdemo
    1
    1. 初始化package.json，执行下面命令。键入后提示输入packagename、description等几项参数，暂时可以不输入直接回车。执行过后目录下出现新文件“package.json”
    npm init
    1
    1. 在package.json文件中的script语句块下添加两行代码
    "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
  //下面两行是要添加的
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  },
    1
    2
    3
    4
    5
    6
    1. 键入下面命令将VuePress安装为本地依赖。安装后文件中生成node_modules文件夹和package-lock.json文件
    npm install -D vuepress
    1
    1. 输入如下命令在项目根目录下新建docs文件夹并，在docs文件夹中建立名为“README.md”的markdown文件，并存入文本“Hello VuePress”
    mkdir docs && echo 'Hello VuePress' > docs/README.md
    1
    1. 启动本地服务器
    npm run docs:dev
    1
    1. 在浏览器打开http://localhost:8080/，显示下图即代表成功在本地启动了一个热重载的开发服务器

    1. 更换主题，这里以主题vuepress-theme-reco为例。在项目根目录下执行命令安装主题
    npm install vuepress-theme-reco --save-dev
    1

    安装成功后，在docs/.vuepress/config.js中添加本行应用主题

    module.exports = {
//下面这行为添加内容
  theme: 'reco'
}
