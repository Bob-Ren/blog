    超好用的markdown编辑器 — Typora（快捷键）

    Bob.Ren 2/8/2022 Typora

    # 文件、窗口相关

    功能 快捷键（Windows） 快捷键（macOS）
    新窗口 Ctrl + Shift + N Command + N.
    新标签 (不支持） Command + T
    打开 Ctrl + O Command + O
    打开刚关闭文件 Ctrl + Shift + T Command + Shift + T
    保存 Ctrl + S Command + S
    另存为 Ctrl + Shift + S Command + Shift + S
    偏好设置 Ctrl + , Command + ,
    关闭窗口 Ctrl + W Command + W

    # 编辑功能

    功能 快捷键（Windows） 快捷键（macOS）
    新段落 Enter Enter
    新行 Shift + Enter Shift + Enter
    复制 Ctrl + C Command + C
    剪切 Ctrl + X Command + X
    粘贴 Ctrl + V Command + V
    复制为markdown Ctrl + Shift + C Command + Shift + C
    粘贴为纯文本 Ctrl + Shift + V Command + Shift + V
    全选 Ctrl + A Command + A
    选择行 Ctrl + L Command + L
    删除行（光标位置之前） Ctrl + Shift + Backspace Command + Shift + Backspace
    选择词 Ctrl + D Command + D
    删除词 Ctrl + Shift + D Command + Shift + D
    选择样式范围 Ctrl + E Command + E
    回到顶部 Ctrl + Home Command +↑
    跳转到光标位置 Ctrl + J Command + J
    查找 Ctrl + F Command + F
    查找下一个 F3 / Enter Command + G / Enter
    查找上一个 Shift + F3 / Shift + Enter Command + Shift + G / Shift + Enter
    替换 Ctrl + H Command + H

    # 段落

    功能 快捷键（Windows） 快捷键（macOS）
    标题1到6 Ctrl + 1/2/3/4/5/6 Command + 1/2/3/4/5/6
    提高标题级别 Ctrl + = 命令+ =
    降低标题级别 Ctrl + - 命令+ -
    表 Ctrl + T Command + Option + T
    代码围栏 Ctrl + Shift + K Command + Option + C
    数学块 Ctrl + Shift + M Command + Option + B
    引用 Ctrl + Shift + Q Command + Option + Q
    订购清单 Ctrl + Shift + [ Command + Option + O
    无序列表 Ctrl + Shift +] Command + Option + U
    缩进 Ctrl + [/ Tab Command + [/ Tab

    # 视图

    功能 快捷键（Windows） 快捷键（macOS）
    切换补充工具栏 Ctrl + Shift + L Command + Shift + L
    大纲 Ctrl + Shift + 1 Command + Control + 1
    文档目录 Ctrl + Shift + 2 Command + Control + 2
    文件目录 Ctrl + Shift + 3 Command + Control + 3
    源代码模式 Ctrl + / Command + /
    Focus模式 F8 F8
    打字机模式 F9 F9
    清屏/恢复 F11 Command + Option + F.
    真实大小 Ctrl + Shift + 0 （不支持）
    放大 Ctrl + Shift + = （不支持）
    缩小 Ctrl + Shift + - （不支持）
    在打开的文档之间切换 Ctrl + Tab Command + `

    # 避免转义

    在符号前加 " \ "
